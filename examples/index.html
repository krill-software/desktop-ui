krill chrome — examples

Live reference for the desktop-ui chrome. These run against ../src directly — no version pin — so they always show the current style. Edit src/styles/*.css and the page reloads.

  • Quiet (single column) titlebar · menu · viewport · status line — text-editor, image-viewer, document-viewer
  • Manipulation (aux pane) + 260px left aux pane — image-editor, color-editor
  • App (sidebar workspace) no titlebar/status; main-topbar + aux hamburger + scroll content — file-drop, audio-editor, paint, color-editor, system-monitor, markdown-viewer, photo-importer
  • Component gallery empty / error states · loader · filter input · kbd
